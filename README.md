

# Reproduction d’une maquette UI
## Description
Ce projet consiste à reproduire une maquette fournie dans le cadre d’un test technique pour un poste de développeur.
Il montre mes compétences en React et Tailwind CSS pour créer une interface responsive et fidèle au design fourni.

# Fonctionnalités

Interface responsive (mobile, tablette, desktop)
Composants React modulaires et réutilisables
Stylage rapide avec Tailwind CSS
